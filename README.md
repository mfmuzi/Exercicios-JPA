

### Exercícios da aula de JPA

#### Parte 1

* Configure uma aplicação JPA de acordo com os passos explicados na aula sem implementação.
* Configure a aplicação desenvolvida com JPA para utilizar o Hibernate e o EclipseLink.

#### Parte 2

* Crie uma consulta SQL e execute diretamente no Banco de Dados.
* Realize a mesma consulta utilizando  JPQL e JPA Criteria API.
